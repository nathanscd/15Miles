# Project: 15Miles - The Space Portal Studio

## Context
15Miles is a tech-focused company. The aesthetic is now a high-contrast "Space Portal" concept: a clean, premium white environment (the spaceship) with "portals" (black divs with stars) that reveal the cosmic tech work.

## Objectives
- Implement a "White & Space" high-contrast design system.
- Use **Raleway** as the primary font for a clean, premium feel.
- Create "Space Portals" (deep black divs with animated starfields) for content.
- Achieve an "entering/exiting space" transition effect.

## Tech Stack
- **Frontend**: React (Vite) + TypeScript
- **Styling**: Vanilla CSS (Modern design system)
- **Animation**: CSS Keyframes + Transitions (Focus on high-contrast portal reveals)

## Brand Identity
- **Keywords**: Portal, High-Contrast, Minimalist, Precision, Discovery.
- **Colors**: Stark White (#ffffff), Deep Space Black (#000000), Stellar Blue (#63b3ff).
