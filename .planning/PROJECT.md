# Project: 15Miles - Interstellar Tech Website

## Context
15Miles is a tech-focused company specializing in building websites and software. The goal is to redesign the website with an "award-winning" space-themed aesthetic that feels beautiful, unique, and premium.

## Objectives
- Implement a comprehensive "Space-Tech" design system.
- Use unusual, futuristic typography (Orbitron, Space Grotesk).
- Create immersive background effects (twinkling stars, nebulae, shooting stars).
- Build a high-performance, multi-page React application.

## Tech Stack
- **Frontend**: React (Vite) + TypeScript
- **Styling**: Vanilla CSS (Modern design system)
- **Routing**: React Router DOM
- **Animation**: CSS Keyframes + Transitions (Focus on "Antigravity" weightlessness)

## Brand Identity
- **Keywords**: Interstellar, Weightless, Precision, Future, Advanced Tech.
- **Colors**: Deep Obsidian (#040408), Electric Blue (#63b3ff), Cosmic Violet (#a78bfa).
